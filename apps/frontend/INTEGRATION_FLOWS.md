## Summary of Key Integration Points

### Frontend â†’ Backend

| Trigger | Endpoint | Data Sent | Expected Response |
|---------|----------|-----------|-------------------|

### Backend â†’ Frontend

| Scenario | Status Code | Response Format | Frontend Action |
|----------|-------------|-----------------|-----------------|

| Validation error | 400 | `{ error: "..." }` | Show error message |
